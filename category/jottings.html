Setting up a blog using Pelican and GitHub Pages

Published: 土 14 2月 2015
By tksn

In Jottings.

tags: python blog pelican github

Installation

Pelican:

pip install pelican

ghp-import:

pip install ghp-import

Creating GitHub Pages

  1. Create GitHub account
  2. Generate SSH Key and register it (optional. otherwise, use https)
  3. Create username.github.io repository as a User Page

Creating and initializing working directory

mkdir workdir
cd workdir
git init
git remote add origin git@github.com:username/username.github.io.git

Working with Pelican

To launch server and allow automatic html generation:

make devserver

This starts pelican and http.server processes on background. To stop them:

./develop_server.sh stop

To publish to GitHub:

make github

See Makefile.
